# Notas para asistentes de IA (Claude Code y Codex)

Este proyecto es **VEO (Vida en Orden)**. Lo trabaja Agustín, que **no programa**: pide cambios en castellano y el asistente los implementa. Ver `README.md` para el concepto del producto.

## ⚠️ Importante: se trabaja con DOS asistentes

Este proyecto se desarrolla con **dos asistentes distintos en la misma computadora**: **Claude Code** y **Codex**. No se ven entre sí. Para que no se pisen el trabajo, GitHub (`github.com/Klausstin/FOHM`) es la única fuente de verdad.

### Regla de oro (la "posta")

1. **Al EMPEZAR una sesión de trabajo:** traé lo último de GitHub (`git pull`) antes de tocar cualquier archivo. Puede haber cambios que dejó el otro asistente.
2. **Al TERMINAR una tanda de cambios:** guardá y subí a GitHub (`git add` + `git commit` + `git push`).
3. **Nunca** asumas que la carpeta está actualizada: si hay dudas, `git pull` primero.

### Trato con Agustín

- Habla en castellano y es principiante. Explicá en criollo, sin tecnicismos.
- Cuando dice "subí los cambios" → hacé commit + push.
- Cuando dice "traé lo último" → hacé pull.
- Avisale en lenguaje simple qué hiciste y por qué.
